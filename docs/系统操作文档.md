# 拍卖系统操作文档

## 1. 登录系统

1. 打开：`/login`
2. 输入用户名和密码
3. 登录后进入后台首页（默认用户管理）

> 默认管理员账号由后端配置 `admin.bootstrap.*` 决定。

## 2. 菜单说明

- 用户管理
- 拍品管理
- 拍卖管理
- 订单管理
- 系统管理

## 3. 用户管理

路径：`/dashboard/user`

### 3.1 个人中心

- 查看当前登录用户信息
- 提交实名认证（真实姓名、身份证号）
- 充值保证金

### 3.2 用户列表与权限

- 新增用户（用户名、密码、昵称）
- 编辑用户（昵称等）
- 删除用户
- 绑定角色
- 快捷设置角色：买家 / 卖家
- 移除用户角色（点击角色标签关闭）

## 4. 拍品管理

路径：`/dashboard/item`

- 发布拍品（标题、描述、分类、起拍价、加价幅度）
- 按关键词/状态搜索拍品
- 管理员审核拍品（通过 / 拒绝）

状态说明：

- `0` 待审核
- `1` 审核通过
- `2` 审核拒绝
- `3` 拍卖中
- `4` 已成交
- `5` 流拍

## 5. 拍卖管理

路径：`/dashboard/auction`

- 查看专场列表
- 快速进入实时竞价页
- 结束拍卖（仅拍卖中可操作）

实时竞价页路径：`/auction/live/{itemId}`

## 6. 订单管理

路径：`/dashboard/order`

- 查看我的订单（买家和卖家相关订单）
- 支付（买家、待支付）
- 发货（卖家、已支付）
- 完成（买家、已发货）
- 取消（待支付且交易相关方）

订单状态：

- `0` 待支付
- `1` 已支付
- `2` 已发货
- `3` 已完成
- `4` 已取消

## 7. 系统管理

路径：`/dashboard/system`

### 7.1 分类管理

- 新增分类
- 编辑分类
- 删除分类

### 7.2 专场管理

- 新增专场
- 编辑专场
- 删除专场

### 7.3 权限管理

- 新增角色
- 新增权限
- 绑定角色到用户
- 绑定权限到角色

### 7.4 操作日志

- 查看系统操作日志
- 支持分页浏览

## 8. 典型业务流程（建议测试）

1. 管理员创建分类、专场
2. 卖家发布拍品
3. 管理员审核通过拍品
4. 买家在实时竞价页出价
5. 卖家结束拍卖，系统自动生成订单
6. 买家支付 -> 卖家发货 -> 买家确认完成

## 9. 权限说明（当前实现）

- 接口鉴权基于登录 token（`Bearer uid:{id}`）
- 管理能力由 RBAC 权限决定（如 `admin:user:list`）
- 超级管理员权限：`admin:*`

## 10. 常见问题

### 10.1 页面操作无反应

- 前端已增加统一错误提示，请查看提示信息
- 常见原因：无权限、参数不完整、重复数据

### 10.2 登录失败

- 用户名密码错误
- 用户被禁用
- 后端或数据库不可用

### 10.3 角色分配后不生效

- 刷新用户列表确认分配结果
- 检查是否绑定了正确角色和权限
