# 一键脚本与生产模板说明

## 1. 脚本目录

```text
scripts/
  start-dev.bat
  start-dev.sh
  start-prod.bat
  start-prod.sh
  stop-prod.bat
  stop-prod.sh
```

## 2. 开发环境一键启动

### Windows

```bat
scripts\start-dev.bat
```

### Linux/macOS

```bash
chmod +x scripts/*.sh
./scripts/start-dev.sh
```

## 3. 生产环境启动

### 3.1 先打包 Jar

```bash
cd auction-backend
mvn clean package -DskipTests
```

### 3.2 配置生产配置文件

复制模板并修改：

```text
auction-backend/src/main/resources/application-prod.yml.template
-> auction-backend/src/main/resources/application-prod.yml
```

### 3.3 启动生产后端

Windows:

```bat
scripts\start-prod.bat
```

Linux/macOS:

```bash
./scripts/start-prod.sh
```

### 3.4 停止生产后端

Windows:

```bat
scripts\stop-prod.bat
```

Linux/macOS:

```bash
./scripts/stop-prod.sh
```

## 4. Nginx 生产模板

路径：

```text
deploy/nginx/auction-prod.conf.template
```

使用方式：

1. 复制为你的站点配置文件
2. 修改 `server_name`
3. 修改 `root` 为前端 `dist` 实际路径
4. `nginx -t` 检查后重载

## 5. 建议

- 生产密码请务必修改模板中的默认值
- 建议配合 systemd / nssm 做后端进程守护
- 建议给 Nginx 配置 HTTPS（证书）
